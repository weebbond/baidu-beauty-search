# baidu-beauty-search
baidu-beauty-search 是一个使用 Python 编写的简单网络爬虫工具，用于从百度搜索引擎获取关于“美白”主题的搜索结果。该工具利用 requests 库进行网页请求，并使用 BeautifulSoup 进行 HTML 内容解析。通过运行该脚本，用户可以获得关于“美白”的最新搜索结果，包括标题、链接和摘要。
